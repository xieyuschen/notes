# Vim Command Note

<Ctrl+ws>  <Ctrl+wv> :open a new window 
<o/v> also could open new window
<Ctrl+wc>: close current window
</Ctrl+w> +/- : height
<Ctrl+w> <> : width
<Ctrl+w _> : set height
<Ctrl+w |> : set width
<Ctrl+w> = : equilize width and height
<n/N> : search next/previous
<Ctrl+->: go back to list from a file in netrw
<:E>, go back to explore if the above one doesn't work
<cn/cp>: next/previous in grep output.
<:! command> : run shell command outside vim
<Ctrl+v> : select a column of text
